# Coffee Shop Sales Analysis

## Project Overview

This project analyzes the sales performance of a coffee shop using
**NumPy** and **Pandas**. It organizes sales data, calculates revenue,
and generates useful business insights in a simple report.

------------------------------------------------------------------------

## Objective

Practice the following concepts:

-   NumPy Arrays
-   Pandas DataFrame
-   Creating New Columns
-   Statistical Functions
-   Business Data Analysis

------------------------------------------------------------------------

## Technologies

-   Python 3
-   Jupyter Notebook / Google Colab

### Libraries Used

-   NumPy
-   Pandas

------------------------------------------------------------------------

## Features

-   Create NumPy arrays for coffee sales data.
-   Build a Pandas DataFrame.
-   Calculate Revenue for each coffee.
-   Find the most sold coffee.
-   Find the most expensive coffee.
-   Calculate average coffee price.
-   Calculate total revenue.
-   Display a final sales report.

------------------------------------------------------------------------

## How to Run

1.  Open the notebook in Jupyter Notebook or Google Colab.
2.  Install NumPy and Pandas if required.
3.  Run all cells.
4.  View the generated sales report.

------------------------------------------------------------------------

## Concepts Covered

-   NumPy Arrays
-   Pandas DataFrame
-   Data Analysis
-   Creating New Columns
-   Sum, Mean, Maximum
-   Business Reporting

------------------------------------------------------------------------

## Learning Outcomes

After completing this project, students will be able to:

-   Create NumPy arrays.
-   Build Pandas DataFrames.
-   Perform basic business analysis.
-   Generate analytical reports using Python.
